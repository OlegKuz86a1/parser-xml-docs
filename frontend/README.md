# Артвелл — система управления исполнительной документацией

Веб-интерфейс для загрузки, валидации и хранения строительных документов в формате XML.

## Быстрый старт

Откройте `index.html` в браузере **через локальный HTTP-сервер** (иначе возможны ограничения для `file://`):

```bash
cd frontend
python3 -m http.server 8080
```

Затем перейдите в браузере на `http://127.0.0.1:8080/`.

## Структура

См. `scheme.txt`. Скрипты подключаются в порядке: `js/data.js` → `js/api.js` → `js/app.js`.

## Бэкенд

`js/api.js` содержит заготовки под REST API (`API_CONFIG.baseUrl`). Пока сервера нет, интерфейс работает на данных из `js/data.js`.

Стартовый экран — **Документы**.
